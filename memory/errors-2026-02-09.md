## Prevention Checklist

Add to every agent prompt:

```
Before marking complete:
- [ ] Test file opens without errors
- [ ] Verify all links/paths work
- [ ] Confirm file is tracked in git (git status)
- [ ] Check mobile responsiveness (if UI)
- [ ] Verify no placeholder data remains
```

## Common Error Patterns

| Error | Cause | Fix | Prevention |
|-------|-------|-----|------------|
| 404 on new page | Not in git | git add + commit | Checklist item |
| Broken links | Hardcoded paths | Use relative paths | Code review |
| Style conflicts | CSS not scoped | Add unique classes | Naming convention |
| Mobile issues | Not tested | Test on small screen | Responsive check |
| Placeholder content | Forgot to replace | Audit before deploy | Pre-deploy checklist |
