Installation

1. Include the CSS and JS files:

<link rel="stylesheet" href="/components/FeedbackWidget.css">
<script src="/components/FeedbackWidget.js"></script>

2. Initialize the widget:

<script>
  FeedbackWidget.init({
    apiUrl: 'https://api.jochero.dev/feedback',
    userEmail: 'user@example.com'  // optional
  });
</script>

Or use auto-init with data attributes:

<body data-feedback-widget 
      data-api-url="https://api.jochero.dev/feedback"
      data-user-email="user@example.com">

API

Configuration Options

{
  apiUrl: '/api/feedback',     // API endpoint URL
  userEmail: '',               // Pre-fill email field
  position: 'bottom-right',    // Button position (future)
  buttonText: 'üêõ',            // Button emoji/text
  maxChars: 5000,              // Max description length
  maxFileSize: 5242880         // Max screenshot size (5MB)
}

Methods

FeedbackWidget.init(options)  // Initialize widget
FeedbackWidget.show()         // Open the modal
FeedbackWidget.hide()         // Close the modal
FeedbackWidget.destroy()      // Remove widget from DOM
