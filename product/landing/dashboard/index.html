üìÑ Uploading Your First Drawing

Drag & drop any PDF onto the main canvas, or click the Upload PDF button in the header.

Supported: PDF files up to 50MB. Multi-page drawings work great!

üìè Setting the Scale

Click the üìê ruler icon in the toolbar to open calibration.

  • Presets: Choose common scales like 1/4" = 1'-0"
  • Measure: Draw over a known dimension on the plan
  • Manual: Enter pixels-per-foot directly
‚úèÔ∏è Your First Measurement
  1. Press L for Line tool
  2. Click the start point on your drawing
  3. Click the end point - measurement appears!
  4. View measurements in the right panel
üì• Exporting to CSV

Click Export CSV in the measurements panel. The file includes all measurements with JOC line items, quantities, and totals - ready for your proposal.

üìè Line Tool (L) - Linear Feet

Click start point ‚Üí click end point. Perfect for walls, pipes, conduit, trim. Shows length in LF.

‚¨õ Area Tool (A) - Square Feet

Click to place polygon points. Double-click to close. Great for flooring, ceiling, paint areas. Shows area in SF.

üî¢ Count Tool (C) - Items

Click to count items (lights, receptacles, fixtures). Each click adds a numbered marker. Double-click to finish.

üè† Space Tool (S) - Rooms

Define room boundaries. Calculates both area AND perimeter. Name each space for finish schedules.

üìã Using Scale Presets

If you know your drawing's scale (check the title block), use presets:

‚Ä¢ 1/8" = 1'-0" (floor plans)
‚Ä¢ 1/4" = 1'-0" (standard)
‚Ä¢ 1/2" = 1'-0" (details)
‚Ä¢ 1" = 10' (site plans)
üìê Manual Calibration
  1. Find a dimension line with a known length (e.g., "10'-0"")
  2. Use the Line tool to trace that dimension
  3. Open Calibration ‚Üí Measure tab
  4. Enter the real distance + measured pixels
  5. Scale is calculated automatically!
üñ•Ô∏è PDF DPI Settings

If scale seems off, try adjusting PDF DPI: 72 (standard), 96 (Windows), 150 (print), 300 (high-res). Most construction PDFs are 72 or 150.

üîß Troubleshooting Scale
  • Measurements too big? Increase pixels-per-foot
  • Measurements too small? Decrease pixels-per-foot
  • Always verify by measuring a known dimension
üìë Organizing Multi-Page Drawings

Use the page navigation panel (left side) to jump between sheets. Thumbnail previews help you find the right page fast. Scale may vary between sheets - recalibrate if needed.

‚ö° Efficient Takeoff Workflow
  1. Set scale first (always!)
  2. Do all measurements of one type together
  3. Use keyboard shortcuts - don't click tools
  4. Assign JOC items in batches at the end
  5. Review totals before exporting
üéØ Accuracy Tips
  • Zoom in for precise clicks
  • Use Snap (‚ö°) to connect measurements
  • Double-check scale with a known dimension
  • Name your spaces clearly for the proposal
üì§ Preparing Your Proposal

Before exporting: verify all measurements have JOC items assigned, check the coefficient/multiplier, review the Summary tab for totals. Export generates a clean CSV ready for your JOC proposal template.
