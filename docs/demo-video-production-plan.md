## Technical Specifications

### Resolution & Format
- **Resolution:** 1920Ã—1080 (Full HD)
- **Frame Rate:** 30fps
- **Duration:** ~3 minutes
- **Export:** MP4 (H.264), WebM (for web)
- **Aspect Ratios:** 16:9 (main), 1:1 (social cut), 9:16 (mobile)

### Color Palette
- **Primary:** #1E40AF (Trust Blue)
- **Secondary:** #1E293B (Dark Navy)
- **Accent:** #FFD700 (Gold)
- **Background:** #0F172A (Deep Navy)
- **Text:** #F1F5F9 (Off-white)
- **Success:** #10B981 (Green)
- **Error:** #EF4444 (Red)

### Typography
- **Headlines:** Inter Bold / SF Pro Bold
- **Body:** Inter Regular / SF Pro Regular
- **UI Text:** System fonts (San Francisco, Segoe UI)
- **Numbers:** Monospace for data (Roboto Mono)

### Animation Timing
- **Easing:** cubic-bezier(0.4, 0, 0.2, 1) for UI
- **Bounce:** cubic-bezier(0.68, -0.55, 0.265, 1.55) for emphasis
- **Smooth:** cubic-bezier(0.25, 0.1, 0.25, 1) for transitions
- **Duration:**
  - Micro-interactions: 200-300ms
  - Screen transitions: 500-700ms
  - Major reveals: 800-1000ms

## Tools & Software

### Option A: After Effects (Professional)
**Best for:** High-end production, smooth animations
- **Pros:** Full control, industry standard, export any format
- **Cons:** Learning curve, subscription ($55/mo)
- **Plugins:** Motion2, Flow (for easing), Overlord (Figma â†’ AE)

### Option B: Figma + Animation Plugins (Easier)
**Best for:** Quick turnaround, iterative design
- **Pros:** Free, collaborative, design + animate in one tool
- **Cons:** Limited animation complexity
- **Plugins:** Figmotion, Anima, Motion

### Option C: Canva + Screen Recording (Fastest)
**Best for:** MVP demo, quick turnaround
- **Pros:** Free, easy templates, fast
- **Cons:** Less professional, limited customization

### Option D: Spline 3D (Unique)
**Best for:** Standout 3D animations
- **Pros:** 3D elements, modern, memorable
- **Cons:** Learning curve, may be overkill

## Voiceover Recording

### Equipment
- **Microphone:** Blue Yeti, Audio-Technica AT2020, or Rode NT-USB
- **Interface:** Focusrite Scarlett Solo (if using XLR mic)
- **Recording Environment:** Quiet room, soft furnishings (reduce echo)

### Tips
- Record in 48kHz, 24-bit WAV
- Use pop filter
- Record multiple takes
- Leave 2-second pauses between lines
- Normalize audio to -3dB
- Add subtle compression

### Voiceover Artist (Optional)
If not using your own voice, hire on:
- Fiverr ($50-200)
- Voices.com ($100-500)
- Bunny Studio ($75-300)

**Style:** Professional, confident, friendly (not sales-y)
